# Changelog

_**2026-03-12**_

Initial commit of new simplified implementation approach:

- dependencies feature to install essential tools and a minimal library for other features.
- non-root user feature to add a non-root user.
- prompt feature to set up a customizable shell prompt with Git integration.
- shell-history feature for managing shell history using atuin and fzf.
- wslg-support feature to configure WSLg and install related utilities.
